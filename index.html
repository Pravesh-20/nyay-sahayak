One-time setup (for Pravesh)

Papa's app needs just one free API key — ₹0/month; Google's free tier covers daily articles and images (no card, no billing):
  1. Sign in at aistudio.google.com/apikey with any Google account → Create API key.
  2. Paste it in ⚙️ Settings above and tap save.
  3. Tap 🎤 once and allow the microphone, add this page to the home screen, and hand the phone to Papa. 🙏

⚙️ Settings

Keys stay in this phone's browser storage only — they are never uploaded anywhere except directly to Anthropic/Google.
